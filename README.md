# wordle-solver
A customized anagram solver specifically for Wordle puzzles

## Setup
- nvm install v17.6.0
- npx create-react-app my-app

## To Run
- cd my-app
- npm start

## To Deploy
- cd my-app
- npm run deploy
- check that gh-pages branch has been updated. Will auto deploy - check Settings > Pages for latest deploy timestamp
